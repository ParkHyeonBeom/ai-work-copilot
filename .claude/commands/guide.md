docs/dev-guide.md에서 지정한 Day의 가이드를 로드하고 요약해줘.

사용법: /guide [Day번호]  (예: /guide 8)

1. `docs/dev-guide.md` 파일을 읽기
2. 해당 Day 섹션을 찾아서 로드
3. 아래 형식으로 요약:

```
📖 Day {N} 가이드: [제목]
━━━━━━━━━━━━━━━━━━━━

🎯 목표:
  [이 Day에서 달성할 것]

📝 작업 목록:
  1. [구체적 작업]
  2. [구체적 작업]

💡 핵심 포인트:
  - [알아야 할 기술적 사항]

📁 생성/수정할 파일:
  - [파일 경로와 설명]
```

4. Day 번호가 없으면 전체 로드맵 요약을 보여줘
5. "진행해줘"라고 하면 해당 Day 가이드에 따라 구현 시작
