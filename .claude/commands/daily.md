현재 프로젝트 상태를 분석하고 오늘의 작업 계획을 알려줘.

1. `git log --oneline -10`으로 최근 커밋 확인
2. 변경된 파일 확인: `git status`
3. 빌드 상태 확인: `mvn compile -q` (에러만 출력)
4. docs/dev-guide.md에서 현재 Day 파악
5. 다음에 해야 할 작업을 구체적으로 알려줘

출력 형식:
```
📅 현재 진행: Day N - [작업명]
✅ 완료: [최근 완료 항목]
🔜 오늘 할 일:
  1. [구체적 작업 1]
  2. [구체적 작업 2]
  3. [구체적 작업 3]
⚠️ 주의사항: [있으면]
```
